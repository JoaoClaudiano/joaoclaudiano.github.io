Política de Privacidade

Última atualização: 26 de fevereiro de 2026

Esta Política de Privacidade descreve como os projetos hospedados em joaoclaudiano.github.io ("nós", "nosso" ou "o site") coletam, usam e compartilham informações quando você os visita.

1. Coleta de Dados

Nosso site é estático e não possui um backend próprio para coletar dados pessoais diretamente. No entanto, utilizamos serviços de terceiros que podem coletar informações anônimas:

  • Google AdSense: Para exibir anúncios relevantes, o Google pode usar cookies e coletar dados de navegação não pessoalmente identificáveis (como tipo de navegador, páginas visitadas).
  • Google Analytics: Para entender como os visitantes interagem com o site, coletamos métricas anônimas como país de origem, tempo na página e páginas mais acessadas.
  • GitHub OAuth (Aplicável ao GitTree): A ferramenta GitTree solicita acesso público à sua conta do GitHub apenas com seu consentimento explícito. Esta autenticação é gerenciada diretamente pela API do GitHub e nós não temos acesso às suas credenciais. Acessamos apenas os dados de repositório públicos que você autorizar, e isso é feito diretamente no seu navegador.

2. Uso das Informações

As informações anônimas coletadas são usadas para:

  • Exibir anúncios mais relevantes (via Google AdSense).
  • Analisar o tráfego do site para melhorar o conteúdo e a experiência do usuário.
  • Funcionalidade de aplicações específicas (como visualizar repositórios no GitTree).

3. Cookies e Tecnologias Semelhantes

Serviços de terceiros, como o Google, utilizam cookies para personalizar anúncios e analisar o tráfego. Você pode gerenciar as preferências de cookies diretamente nas configurações do seu navegador.

4. Seus Direitos

Você pode recusar a coleta de dados analíticos pelo Google Analytics utilizando a extensão oficial de opt-out do Google. Para controlar os anúncios personalizados, visite as Configurações de Anúncios do Google.

5. Alterações a Esta Política

Podemos atualizar esta política periodicamente. A data da última revisão será sempre indicada no topo desta página.

6. Contato

Se tiver dúvidas sobre esta Política de Privacidade, você pode abrir uma issue no repositório relevante no GitHub ou consultar os termos dos serviços de terceiros mencionados.
