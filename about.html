Sobre

Olá! Meu nome é João Claudiano. Sou um desenvolvedor apaixonado por criar ferramentas práticas e úteis, especialmente voltadas para a comunidade de programação.

Este site no GitHub Pages serve como um hub central para meus projetos de código aberto. Aqui, você pode acessar e experimentar diretamente as aplicações que desenvolvo.

Meu Objetivo

Construir ferramentas que simplificam tarefas do dia a dia de outros desenvolvedores, aprendendo e compartilhando conhecimento no processo. Acredito no poder do código aberto para colaboração e melhoria contínua.

Projetos em Destaque

  • GitTree: Uma ferramenta de visualização interativa de árvores de repositórios do GitHub. Permite navegar pela estrutura de diretórios de um repositório público de forma mais clara e intuitiva.
  • HTML Teste: Um ambiente de testes e exemplos rápidos para conceitos de HTML, CSS e JavaScript. Útil para prototipagem e experimentação.

Tecnologias

Trabalho principalmente com a stack web moderna: JavaScript, HTML, CSS, e frameworks como React. Os projetos neste site muitas vezes interagem com APIs públicas, como a API do GitHub.
